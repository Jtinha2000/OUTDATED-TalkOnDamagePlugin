# Plugin HealWeapon

> Esse repositório contém uma extensão desenvolvida para o jogo Unturned. Apartir dela o player emite sons pré-configurados ao ser atingido ou quebrar a perna.

## ⚠️ Aviso

O projeto foi desenvolvido em 2021 para auxiliar meu aprendizado com a linguagem C#, não espere um código limpo e totalmente funcional. O uso dele é de total risco e responsabilidade do usuário.
